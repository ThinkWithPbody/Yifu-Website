Automate the boring stuff... with JavaScript

What?

I keep track of my favourite Blender addons and parameters for note keeping and easier migration. Twist: I innitially wrote this long list in Markdown. It's not a problem, really, except that now I want it in HTML. So, I wrote a script to convert this long list from Markdown to HTML.

Why?

For this project, filling everything out by hand wouldn't have taken much longer than writing the script, but it's more fun which means I'll actually get it done. But this is just the beginning...

How?
